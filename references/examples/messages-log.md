# Project Messages Log

This log lives in the user project folder and is used for inter-employee messages.

## Format
- Time: ISO timestamp
- From: model id or user
- To: manager or specific employee
- Topic: short label
- Body: message text

## Example
2026-02-10T11:00:00Z | model-002 | manager | Approval request | Please approve task-102 start.
2026-02-10T11:02:00Z | manager-001 | model-002 | Approval grant | Approved to start task-102.
