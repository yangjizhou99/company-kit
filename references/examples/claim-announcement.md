# Task Claim Announcement Template

**Use the strict format below in your employee log.** Do not use free-text claim formats.

```
[<ISO timestamp>] [CLAIM] task-<id> | title: <title> | by: <employeeId> | difficulty: <N> | deps-met: yes | board-read-at: <ISO timestamp>
```

## Example
```
[2026-02-11T10:00:00Z] [CLAIM] task-004 | title: Home Page | by: emp-copilot-a3f2 | difficulty: 3 | deps-met: yes | board-read-at: 2026-02-11T09:55:00Z
```
