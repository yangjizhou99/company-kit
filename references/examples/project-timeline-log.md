# Project Timeline Log

This log lives in the user project folder and records every task claim and delivery.

## Format
- Time: ISO timestamp
- Actor: model id or user
- Event: Claimed | Delivered | Reviewed | Done | Blocked
- Task: task id
- Note: short summary

## Example
2026-02-10T10:00:00Z | model-002 | Claimed | task-101-init-project | Starting project scaffold
2026-02-10T10:25:00Z | model-002 | Delivered | task-101-init-project | Basic HTML/CSS/JS scaffold ready
